# MiTo

**Your money, speaking your language.**

MiTo is an AI-powered personal finance app that lets you manage all your bank accounts, send money, and understand your spending through natural language — in Italian or English.

Built in a weekend at **Buildathon Turin 2026**, organised by [Fonderia](https://www.fonderia.dev/) and sponsored by [Alpic](https://alpic.ai/) and [Manufact](https://manufact.com/).

```bash

```

## Deploy

The project is configured for Vercel. Connect the repo and it deploys automatically — no build step required.
